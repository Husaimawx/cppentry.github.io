Studio是什么？</h6><p>        Visual Spark Studio™是一个免费的、紧凑版本的StreamAnalytix平台，一个轻量级的开发工具，Visual SparkStudio为您提供全面的数据处理和分析功能，在您的桌面或任何单一的节点中创建、测试和运行Spark应用程序。</p><p></p><h6>为什么使用 Visual Spark Studio？</h6><p>        无论是对新的用户还是经验丰富的用户，Visual Spark Studio使得构建Spark数据管道或应用程序变得更容易。用拖放UI操作和内置的Spark操作器和连接器，在分钟级别内构建批处理和流式应用程序。可运行在您的安全本地环境中，不用依赖于其他系统。如果需应用多节点Spark集群的大规模数据管道，一个可选项是使用StreamAnalytix  Visual Spark Studio企业版。</p><p><span style="font-weight:bold;"> </span></p><p><strong>Visual Spark Studio的特点：</strong></p><p>   <strong> 端到端应用程序生命周期管理。</strong></p><p>    VisualSpark Studio是易于使用的可视化集成开发环境（IDE），用来构建、测试、调试、部署和管理基于Apache Spark 的企业级应用程序。</p><p>    <strong>不需要编码，但需启用自定义逻辑。</strong></p><p>    使用拖放操作数据源、sink、transformation转换操作，使用Java和Scala算子自定义逻辑进行分析。</p><p>    <strong>访问数据源广泛。</strong> </p><p>    VisualSpark Studio可访问数据源广泛，数据处理以后可写入HDFS, Kafka, RDBMS,或Solr系统。</p><p><strong>    高级分析与机器学习操作。</strong></p><p>    用于建立预测模型，Spark MLlib, ML及PMML模型移植。</p><p>    <strong>多租户技术。</strong></p><p>    VisualSpark Studio是一个基于Web的平台，多个用户可以连接到一个 Visual Spark Studio实例。</p><p><strong>    平台。</strong></p><p>    可用于Windows、Linux和Mac系统。</p><p></p><h6>关于StreamAnalytix</h6>    StreamAnalytix使企业能够分析和应对在大数据规模的实时事件。它为企业提供了基于开源、企业级平台的优势，可以快速方便地开发适用于任何行业、任何数据格式和任何用例的实时流分析应用程序。<br><p><br></p><h6>StreamAnalytix
