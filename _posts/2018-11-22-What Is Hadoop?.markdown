                <p>What Is Hadoop?<br><br>
The Apache Hadoop project develops open-source software for reliable, scalable, distributed computing. Hadoop includes these subprojects:<br><br>
    * Hadoop Common: The common utilities that support the other Hadoop subprojects.<br>
    * Chukwa: A data collection system for managing large distributed systems.<br>
    * HBase: A scalable, distributed database that supports structured data storage for large tables.<br>
    * HDFS: A distributed file system that provides high throughput access to application data.<br>
    * Hive: A data warehouse infrastructure that provides data summarization and ad hoc querying.<br>
    * MapReduce: A software framework for distributed processing of large data sets on compute clusters.<br>
    * Pig: A high-level data-flow language and execution framework for parallel computation.<br>
    * ZooKeeper: A high-performance coordination service for distributed applications.<br><br><span id="result_box" class="long_text"><span>Hadoop的是什么？<br><br></span>

<span style="background-color:#ffffff;">* Chukwa：一个管理大型分布式系统的数据收集系统。<br>
